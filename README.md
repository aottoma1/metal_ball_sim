
# Metal Ball Collision Simulator

OpenGL-based physics simulator using Euler's Integration


## Dependencies


MacOS
```bash
brew install cmake glfw
```

Linux
```bash
sudo apt install cmake libgl1-mesa-dev libglfw3-dev
```
## Build
```bash
git clone https://github.com/aottoma1/metal_ball_sim.git
cd metal_ball_sim
mkdir build && cd build
cmake ..
make
```
## Textures
To work properly, the textures folder must be moved from the root folder into the created build folder.
## Run
```bash
./MetalBallSim
```
